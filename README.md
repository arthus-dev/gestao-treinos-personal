# 🏋️‍♂️ Sistema de Gestão de Treinos Personalizados

Este é um sistema web responsivo desenvolvido para **Personal Trainers** gerenciarem seus alunos e enviarem treinos de forma rápida e prática diretamente pelo WhatsApp, utilizando histórico de dados persistido localmente.

> 🌐 **Acesse o projeto rodando ao vivo aqui:** [https://arthus-dev.github.io/gestao-treinos-personal/](https://arthus-dev.github.io/gestao-treinos-personal/)

---

## 📱 Funcionalidades

- **Cadastro de Alunos:** Adicione e gerencie dados básicos dos alunos.
- **Montagem de Treinos:** Interface intuitiva para listar exercícios, séries, repetições e cargas.
- **Envio Direto via WhatsApp:** Gera uma mensagem perfeitamente formatada e abre o WhatsApp com um clique para enviar o treino ao aluno.
- **Histórico Local:** Utiliza `localStorage` do navegador para manter as informações salvas mesmo após fechar a página.
- **Visual Premium (Dark Mode):** Interface moderna com tema escuro e detalhes em verde esmeralda para melhor usabilidade.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da aplicação.
- **CSS3:** Estilização moderna com variáveis, Flexbox/Grid para responsividade (mobile-friendly) e paleta de cores personalizada.
- **JavaScript (ES6+):** Lógica de manipulação de dados, persistência local e integração de envio com a API do WhatsApp.
